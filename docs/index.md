# 首页

## 简介

主要是使用编程做一些小工具，做信息整合与分析程序。

## 应用

### DAC

一个数据处理程序的框架，按步骤执行分析过程，对过程进行配置、结果进行交互。

### DAC web

DAC的网页版，使用相同的方式进行数据分析步骤。
布署成网页版方便远程分析、简易访问，过程分享与结果推送。

### MILation

一个存储七大姑八大姨、远房亲戚小孩是谁的单机版网页应用。

### Hashare-Offline

一个纯手工Javascript做的网页版列表编辑与显示应用。用于给自己添加些带时间戳的想法。

### _TPL_

测试计划与记录的工具。

### _TimeLine_

手动维护一些事迹的时间线，并且能合并在一起看看当时的其他人在忙什么。

### A-HotKey

事实上，最有用的一个是在Windows下的快捷键映射工具。
